# Projekt pri predmetu RV - robotski vid za mobilno platformo IHS

## How to run
python   "file"    "path_to_data"     "options"
python .\main.py .\data_rv_ljudje\ --detect-humans

- pip freeze > requirements.txt

- python -m venv .venv
- .venv\Scripts\activate
